# Capture V2 Backlog

This backlog documents intentionally deferred scope after shipping the functional Delivery OS thin slice.

## Core Gaps vs Full Blueprint

1. Offer Engine depth
- Need dedicated Offer CRUD UI and ROI calculator-lite view.
- Need offer-to-pilot conversion analytics.

2. Delivery Engine automation
- Need execution checklist templates per pilot.
- Need richer pilot lifecycle events and SLA tracking.
- Need n8n / external workflow linkage fields with status sync.

3. Conversation lifecycle depth
- Need conversation edit/history timeline UI.
- Need explicit objection taxonomy and outcome tracking.

4. Reporting depth
- Add selectable time windows to dashboard (7/30/custom).
- Add conversion funnel ratios (opened -> fit -> proposed -> active -> retained).

5. Learning Loop instrumentation
- Capture failed call reasons and broken payload incidents.
- Weekly summary endpoint for “lesson -> fix -> outcome” cycle.

## Proposed V2.1 Task Candidates

1. Add Offer CRUD + ROI card + link offer to lead/pilot in UI.
2. Add conversation timeline view with edit/update and objection tags.
3. Add weekly learning report endpoint + dashboard widget.
